# Khumbuka V1.5

Restaurant Profitability Intelligence & Cost Management Platform.

## Setup

1. Open Supabase SQL Editor.
2. Run `supabase/sql/khumbuka_v15_schema.sql`.
3. Deploy/open `index.html`.
4. Use Settings to confirm brands and branches.
5. Add ingredients, recipes, products, sales and expenses.

## Main Flow

Ingredients → Recipes → Products → Pricing → Sales Import → Profit Analysis → Expense Analysis → Company Profitability

## Important

This is not a POS, billing, inventory, purchase or supplier system. Petpooja remains the operational system. Khumbuka is the profitability layer.
